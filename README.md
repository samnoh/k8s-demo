# Kubernetes

## `TIL`

### Basic Commands

-   Read

```bash
kubectl get pods
kubectl get deployments
kubectl get services
kubectl get secrets
kubectl get pvc
```

-   Apply changes

```bash
kubectl apply -f test.yaml
kubecll apply -f k8s
```

-   Delete Config

```bash
kubectl delete deployments test-deploy.yaml
kubectl delete -f k8s
kubectl delete secrets aws_access_key
```
